# OSINT Investigation Workspace (HTML)

Professional OSINT investigation workspace to document research steps, track evidence, manage tasks, generate advanced Google operators, and export case-ready reports.

## Download

1. Download `OSINT Investigation Workspace.html.zip` from this repository.
2. Extract the ZIP file.
3. Open `OSINT Investigation Workspace.html` in your browser.

No installation required.

## What It Includes

- Investigation timeline with editable entries.
- Per-step notes, source reliability, and confidence fields.
- Attachment support with automatic SHA-256 integrity hashing.
- Next-actions task panel and custom checklist.
- Flowchart studio for lead-based OSINT pivots.
- Advanced Google Operators lab with customizable query generation.
- Export options: JSON, TXT, Print/PDF, and folder/ZIP package with evidence mapping.

## Recommended Usage

1. Start by filling case briefing details.
2. Log each action as soon as it is performed.
3. Attach supporting screenshots/files per step.
4. Keep notes factual and separate assumptions from verified findings.
5. Use Flowcharts and Google Operators to expand leads methodically.
6. Export regularly to preserve an audit-ready trail.

## Browser Notes

- Works in modern browsers (Chrome, Edge, Firefox, Safari).
- Some export features may be more complete in Chromium-based browsers.

## Data Handling

- Data is stored locally in browser storage.
- Attachments are managed via local browser storage mechanisms.
- Keep regular exports as backup copies.
